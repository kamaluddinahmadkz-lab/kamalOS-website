Step 1: Download the ISO

Click the "Download kamalOS 1.0 ISO" button above to download kamalOS-1.0.iso (approximately 3-6 GB)

Save it to your Downloads folder

Step 2: Download VirtualBox

Download and install VirtualBox (free virtualization software):

Download VirtualBox

Choose the version for your operating system (Windows, macOS, or Linux)

Step 3: Create a Virtual Machine

  1. Open VirtualBox
  2. Click "New" to create a new VM
  3. Name: kamalOS
  4. Type: Linux
  5. Version: Ubuntu (64-bit)
  6. Memory: 4 GB (4096 MB) or more
  7. Hard disk: Create a virtual hard disk now (25 GB)
  8. Click "Create"

Step 4: Attach the ISO

  1. Select your kamalOS VM in VirtualBox
  2. Click "Settings"
  3. Go to "Storage"
  4. Click the empty CD icon under "Controller: IDE"
  5. Click the CD icon on the right ‚Üí "Choose a disk file"
  6. Select your downloaded kamalOS-1.0.iso
  7. Click "OK"

Step 5: Start kamalOS

  1. Select your kamalOS VM
  2. Click the green "Start" button
  3. The VM will boot from the ISO

‚ö†Ô∏è Don't panic! You'll see "Ubuntu" during the boot process. This is completely normal - Ubuntu is part of kamalOS. The customization happens after installation.

Step 6: Choose "Try or Install Ubuntu"

When you see the boot menu:

  1. Select "Try or Install Ubuntu"
  2. Press Enter
  3. Wait for it to load (this takes 1-2 minutes)

Step 7: Install Ubuntu (which becomes kamalOS)

  1. Click "Install Ubuntu" on the desktop
  2. Choose your language
  3. Select "Normal installation"
  4. Check "Download updates while installing"
  5. Choose "Erase disk and install Ubuntu"
    (Don't worry, this only affects the virtual disk, not your real computer)
  6. Create your username and password
  7. Wait for installation (10-20 minutes)
  8. When prompted, click "Restart Now"

üéâ Step 8: Welcome to kamalOS!

After installation completes and the system reboots:

  1. VirtualBox will automatically remove the ISO
  2. The system will reboot
  3. Log in with the username and password you created
  4. You're now running kamalOS!
