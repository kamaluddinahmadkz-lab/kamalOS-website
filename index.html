📥 How to Install

1

Download the ISO

Click the download button above to get kamalOS-lite-1.0.iso. Save it to your Downloads folder.

2

Download VirtualBox

Download and install VirtualBox — free software that lets you run kamalOS Lite inside your current OS.

Download VirtualBox →
3

Create a Virtual Machine

  1. Open VirtualBox and click New
  2. Name: kamalOS Lite 1.0
  3. Type: Linux → Version: Ubuntu (64-bit)
  4. Memory: 2 GB (2048 MB) or more
  5. Hard disk: 20 GB (create new)
  6. Click Create
4

Attach the ISO

  1. Select your kamalOS Lite VM → click Settings
  2. Go to Storage
  3. Click the empty CD icon → Choose a disk file
  4. Select your downloaded kamalOS-lite-1.0.iso
  5. Click OK
5

Boot & Install

  1. Click the green Start button
  2. Select Try or Install Lubuntu and press Enter
  3. Click Install Lubuntu on the desktop
  4. Choose Normal installation
  5. Select Erase disk and install (only affects virtual disk)
  6. Create your username and password
  7. Wait 10–15 minutes for installation to complete
⚠️ Don't panic! You'll see "Lubuntu" during boot — that's completely normal. kamalOS Lite is built on top of Lubuntu.

🎉 Step 6 — Welcome to kamalOS Lite 1.0!

  1. After installation, click Restart Now
  2. VirtualBox will automatically remove the ISO
  3. Log in with the username and password you created
  4. You're now running kamalOS Lite 1.0!
✨ Enjoy your beautiful kamalOS Lite system with macOS Tahoe design, custom icons, and smooth performance on older hardware!
2

Install QEMU

Install QEMU on your system using the command below.

On Ubuntu/Debian:

sudo apt install qemu-system-x86 qemu-utils

On Windows:

Download QEMU for Windows →
3

Create a Virtual Disk

Open a terminal and run this command to create a 20GB virtual hard disk:

qemu-img create -f qcow2 kamalOS-lite.qcow2 20G
4

Boot the ISO

Run this command to boot kamalOS Lite from the ISO:

qemu-system-x86_64 \
-m 2048 \
-cdrom kamalOS-lite-1.0.iso \
-hda kamalOS-lite.qcow2 \
-boot d \
-enable-kvm \
-cpu host \
-smp 2
⚠️ Remove -enable-kvm if you are on Windows or get an error about KVM.
5

Install Lubuntu

  1. Select Try or Install Lubuntu and press Enter
  2. Click Install Lubuntu on the desktop
  3. Choose Normal installation
  4. Select Erase disk and install (only affects virtual disk)
  5. Create your username and password
  6. Wait 10–15 minutes for installation to complete
  7. Click Restart Now when done
6

Boot Without the ISO

After installation, boot from the virtual disk instead of the ISO:

qemu-system-x86_64 \
-m 2048 \
-hda kamalOS-lite.qcow2 \
-enable-kvm \
-cpu host \
-smp 2

🎉 Welcome to kamalOS Lite 1.0!

  1. Log in with the username and password you created
  2. You're now running kamalOS Lite 1.0 on QEMU!
✨ Enjoy your beautiful kamalOS Lite system with macOS Tahoe design, custom icons, and smooth performance on older hardware!
