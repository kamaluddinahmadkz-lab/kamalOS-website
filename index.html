📥 How to Install

1

Download the ISO

Click the download button above to get kamalOS-1.1.iso (approximately 5-6 GB). Save it to your Downloads folder.

2

Download VirtualBox

Download and install VirtualBox — free software that lets you run kamalOS inside your current OS.

Download VirtualBox →
3

Create a Virtual Machine

  1. Open VirtualBox and click New
  2. Name: kamalOS 1.1
  3. Type: Linux → Version: Ubuntu (64-bit)
  4. Memory: 4 GB (4096 MB) or more
  5. Hard disk: 25 GB (create new)
  6. Click Create
4

Attach the ISO

  1. Select your kamalOS VM → click Settings
  2. Go to Storage
  3. Click the empty CD icon → Choose a disk file
  4. Select your downloaded kamalOS-1.1.iso
  5. Click OK
5

Boot & Install

  1. Click the green Start button
  2. Select Try or Install Ubuntu and press Enter
  3. Click Install Ubuntu on the desktop
  4. Choose Normal installation
  5. Select Erase disk and install (only affects virtual disk)
  6. Create your username and password
  7. Wait 10–20 minutes for installation to complete
⚠️ Don't panic! You'll see "Ubuntu" during boot — that's completely normal. kamalOS is built on top of Ubuntu.

🎉 Step 6 — Welcome to kamalOS 1.1!

  1. After installation, click Restart Now
  2. VirtualBox will automatically remove the ISO
  3. Log in with the username and password you created
  4. You're now running kamalOS 1.1!
✨ Enjoy your beautiful kamalOS system with macOS Tahoe design, blur effects, magic lamp animations and more!
