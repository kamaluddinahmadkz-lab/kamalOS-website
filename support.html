Common Issues

  • Boot Issues - Make sure to select kamalOS from the GRUB menu
  • Theme Not Applying - Log out and log back in after installing themes
  • Dock Not Showing - Enable Dash to Dock extension in Extension Manager
  • Blur Effects - Enable "Blur my Shell" extension for glassy effects

System Requirements

  • 64-bit processor (x86_64)
  • 4 GB RAM minimum (8 GB recommended)
  • 25 GB free disk space
  • Graphics card with OpenGL support
